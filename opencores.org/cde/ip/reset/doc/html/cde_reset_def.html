Description

This module provides a means to expand and resynchronise a reset signal. It is the basic building block for a synchronous reset distribution tree































































Theory of Operation

This module is the building block used to build a synchronous reset distribution tree. It provides a means to resync local resets and combine them with a syst em wide reset. Each reset consists of a pair of signals. One is the active low reset_n that can drive the asynchronous reset port of a flipflip and the other is the active high reset that can drive anything BUT the asynchronous reset port of a flipflop. The local synchronous reset(s) are resynced to ensure that everything is glitchfree. If they are sourced from an asynchronous clock domain then at least two s tages of resync logic must be provided to prevent metastability issues.
